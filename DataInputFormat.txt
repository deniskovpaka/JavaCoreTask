-TxtFormat.txt- /Пассажирский, Грузовой, Спецназначения/

1) Passenger/Freight/Haulage  - тип поезда “String”
2) Тип двигателя - “int”
3) Уровень комфорт - “int” (в грузовом его нет)
4) Вместимость багажа - “double”
  4.1) количество пассажиров/вид перевозимого товара/сила тяги - “int”
5) количество свободных мест/---------/стоимость перевозки/ - “int”
6) ---------/---------/на ремонте :) - “boolean” (только для спецназначения)

Примеры:
TxtFormatPassengerTrain.txt:
1) Passenger
2) 0 (Diesel)
3) 2 (Luxury)	<<<----- BEGIN ---- Параметры для вагона 
	3.1) 50
	3.2) 25	<<<----- END ---- Параметры для вагона 
4) 5
Всего 6 параметров: 3 для пассажирского вагона и 3 для пассажирского поезда

TxtFormatFreightTrain.txt:
1) Freight
2) 0 (Diesel)
3) 500		<<<----- BEGIN ---- Параметры для вагона
4) 0 (Liquid)	<<<----- END ---- Параметры для вагона
Всего 4 параметра: 2 для грузового вагона и 2 для грузового поезда

TxtFormatHaulageTrain.txt:
1) Haulage
2) 0 (Diesel)
3) 0 (Standart)	<<<----- BEGIN ---- Параметры для вагона
	3.1) 555 <<<----- END ---- Параметры для вагона
4) 100
5) true
Всего 6 параметров: 2 для спец вагона и 4 для поезда

--------- Означает отсутствие этого параметра для конкретного поезда
